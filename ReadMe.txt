This Repository is intended as a way for my friends and I to collaborate on workout plans.

This repository will contain the following in the top level directory:

Stats:
    Will contain whichever statistics about our workouts that we care to track.        
Workouts
    Will contain the collected workouts.
    For more information on the workout formatting see 'Template.md' in the 'Workouts' directory
ReadMe.txt
    Figure it out
